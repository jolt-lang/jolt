# jolt.host surface manifest — the canonical set of names bound into the
# jolt.host namespace via (def-var! "jolt.host" ...) across host/chez/.
# The compiler and the clojure.core overlay reach these by late-bound
# var-deref, so a typo or a removed def would surface only as a runtime
# unbound-var. host/chez/manifest-check.sh (the `manifestcheck` gate) pins
# this list against the def-var! sites and against every jolt.host reference
# in jolt-core, so any drift fails CI. Regenerate after an intended change:
#   grep -rhoE --include='*.ss' 'def-var! "jolt.host" "[^"]+"' host/ \
#     | sed -E 's/.*"jolt.host" "([^"]+)".*/\1/' | LC_ALL=C sort -u
